# `--placeholder` Flag

This speficies that this text node should be interpreted as placeholder of a parent text input element.

## Syntax

```ts
`--placeholder` | `--placeholder=${typeof boolean}` | `--is-placeholder`;
```

## Example

```
--placeholder
--placeholder=true
--placeholder=false
--is-placeholder
```
